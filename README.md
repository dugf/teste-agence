# Teste Agence
<br>
Desenvolvimento de um Aplicativo Flutter com Login através do Google e Facebook; acesso ao Google Maps e atualização local; e consumo de dados json. 
<br>
O aplicativo possui três telas principais:
Login, com acesso pelas redes sociais.<br>
Pedidos, com uma lista de tênis.<br>
Detalhes do Pedido, com a informação do produto selecionado em Pedidos. <br>
Ainda tem o Drawer para navegação e logout.
